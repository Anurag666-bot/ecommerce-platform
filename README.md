# 🛒 Ecommerce Platform

A full-stack ecommerce web application built with Next.js, Node.js, and MongoDB.  
This project includes authentication, product management, cart system, orders, and admin features.

---

## 🚀 Features

- User authentication (login/signup)
- Product listing and search
- Product details page
- Shopping cart system
- Order checkout flow
- User profile management
- Admin product management
- Reviews and ratings system
- Wishlist functionality

---

## 🛠 Tech Stack

- Frontend: Next.js, React, Tailwind CSS
- Backend: Node.js API routes (Next.js API)
- Database: MongoDB (Mongoose models)
- Authentication: NextAuth / JWT
- Deployment: Vercel ready

---

## 📂 Project Structure

- `/app` - Pages and API routes
- `/components` - UI components
- `/utils` - Database and helper functions
- `/store` - State management
- `/public` - Images and assets

---

## 🔒 Security Notes

This project is intended for learning and testing purposes.  
Basic security practices included:
- Password hashing
- Protected API routes
- Input validation
